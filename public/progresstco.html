TCO Report API (1.0.0)

API untuk menghasilkan laporan TCO dengan memproses data dari database Oracle dan mengunduh laporan dalam format Excel dan zip.

Schemas

Generate TCO report

get/getreporttco

Mendapatkan laporan TCO berdasarkan parameter yang diberikan dan mengembalikan file zip dengan laporan.

Responses

Get all TCO jobs

get/jobsdco

Mendapatkan daftar semua pekerjaan TCO yang sedang diproses, beserta statusnya.

Responses

Get job details by ID

get/jobstco/{id}

Mendapatkan detail pekerjaan TCO berdasarkan ID pekerjaan.

Responses

Download TCO report by job ID
